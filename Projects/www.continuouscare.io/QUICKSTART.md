# Quick Start Guide

## Frontend (React.js)

```bash
cd frontend
npm run dev
```

🌐 **Access at:** http://localhost:5173

## Backend (FastAPI)

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

🚀 **API at:** http://localhost:8000  
📖 **API Docs:** http://localhost:8000/docs
